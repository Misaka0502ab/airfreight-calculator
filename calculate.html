列表操作方法

方法 功能说明 示例代码
append() 在末尾添加一个元素 my_list = [1, 2, 3]
my_list.append(4)
print(my_list) # 输出: [1, 2, 3, 4]
insert() 在指定位置插入一个元素 my_list = [1, 2, 3]
my_list.insert(1, 10)
print(my_list) # 输出: [1, 10, 2, 3]
pop() 删除并返回指定位置的元素 x = [1, 2, 3]
y = x.pop()
print(x) # 输出: [1, 2]
print(y) # 输出: 3
sort() 原地排序，默认升序 my_list = [3, 1, 2]
my_list.sort()
print(my_list) # 输出: [1, 2, 3]

编码版使用指南

为什么要用编码版？

  • 实现影刀暂不支持的功能
  • 已经有大量的自动化脚本
  • IT 喜欢敲代码

编码版功能

  • 左侧是影刀开放的指令接口：扩展指令、package、xbot
  • 接口说明文档可以直接查看或跳转到帮助中心

重要注意事项

  1. 编码版调用的是自定义指令
  2. 自定义指令必须发版才能在市场找到，且每次更新都得发版一次，然后在市场更新一次
  3. 自定义指令的主流程对应main，子流程对应一系列process，这里要在子流程里写
  4. 自定义指令如果需要输入与输出，必须在下方的流程参数里新建参数，然后在流程中引用
  5. 自定义指令界面右侧的眼睛可以禁用流程，如果要在编码版中使用某一流程，就不要在此处禁用它

API 调度

HTTP 请求流程

分类 拆解 说明
HTTP 请求流程 客户端行为 客户端向服务器发送请求
HTTP 请求流程 服务器响应 服务器返回对应数据（如 HTML 文档、JSON 数据）

HTTP 请求构成

部分 说明 示例/特点
请求行 - 方法 GET、POST 等 GET 特点：查询数据
POST 特点：提交数据
请求头 - 认证信息 Authorization（鉴权字段） 用于校验接口权限
请求体 主要用于 POST 请求 支持格式：JSON、表单数据

影刀 API 特点

  • 数据格式统一：发送/返回均为 JSON 格式
  • 流程简化：比浏览器请求更简洁，专注 API 交互

JavaScript 函数参数引号规则

参数类型 是否需要引号 示例（正确写法） 错误示例及原因
字符串（文本） 必须加单/双引号 alert("你好！")
console.log('123')		 alert(你好) → 报错：变量未定义
数字（Number） 不加引号 const num = 100
Math.max(5, 10)		 Math.max("5", "10") → 按字符串比较而非数字
布尔值（Boolean） 不加引号 let isOk = true
if (false) { ... }		 if ("true") { ... } → 字符串 "true" 为真，逻辑错误
变量/对象/函数 不加引号 const user = { name: "A" };
console.log(user)		 console.log('user') → 输出字符串 'user' 而非变量
